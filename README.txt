Student Management System - SQL Project

This project demonstrates a basic Student Management System using SQL.

How to Use:
1. Run the SQL script `student_management.sql` in any SQL environment like MySQL, PostgreSQL, or SQLite.
2. It will create two tables: Departments and Students.
3. Inserts sample data into both tables.
4. Executes useful SELECT queries including joins, filters, and aggregation.

Features Demonstrated:
- CREATE TABLE, INSERT INTO
- INNER JOIN
- WHERE, GROUP BY, ORDER BY
- COUNT(), AVG()

Perfect for learning SQL basics with practical use-cases.
